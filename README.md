# BorderLayout

## Description
A Java Swing application demonstrating the use of BorderLayout.

## Requirements
- Java 8 or higher
- IntelliJ IDEA

## Installation
1. Clone the repository
2. Open the project in IntelliJ
3. Run `Main.java`

## Usage
- **NORTH** : Application title
- **WEST** : Three option buttons
- **CENTER** : Scrollable text field
- **SOUTH** : Submit button

## Testing
Run the tests in `MainFrameTest.java` using JUnit.
